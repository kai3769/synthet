# Study

Analyze a repo and save a structured profile to memory.

## Process

1. Find the repo path
2. Read: README, package manifest, directory structure, recent commits, key source files
3. Generate profile: purpose, tech, architecture, key files, relations, activity
4. Save to repos/<org>/<name>.md
5. Commit and push
