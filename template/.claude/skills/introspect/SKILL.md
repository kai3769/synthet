# Introspect

Examine yourself from multiple perspectives to find blind spots.

## Process

1. Spawn parallel critic agents examining:
   - Memory quality (is it accurate, useful, lean?)
   - Skill effectiveness (which skills actually get used?)
   - Communication quality (are messages helpful or noisy?)
2. Synthesize findings into actionable improvements
3. Apply safe improvements, propose uncertain ones to user
4. Update scars.md with any new failure patterns discovered
